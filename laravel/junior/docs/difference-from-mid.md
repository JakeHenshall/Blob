# How this version differs from the Mid version

This junior version focuses on Laravel fundamentals and clear CRUD implementation.

It includes:

- standard MVC structure
- Form Requests for validation
- Eloquent relationships
- basic feature tests
- simple dashboard and filtering
- straightforward Blade views

It intentionally does not include:

- authorisation policies
- role-based access control
- action classes
- queued jobs
- notifications
- audit logging
- API endpoints
- advanced architecture notes
- production trade-off documentation

The goal of this version is to prove strong practical fundamentals and
framework understanding, not advanced system design.
